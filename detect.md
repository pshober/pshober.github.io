This page will host an interactive viewer for **MetDetect** — upload an
automatically detected radar fall file and explore the flagged debris gates,
reflectivity sweeps, and search footprint in your browser.

Because this site is served as static pages, the interactive tool itself will run
on a free companion service (e.g. a Hugging Face Space or Streamlit app) and be
embedded here once it's ready.

In the meantime:

- See the project overview on the [Research](/research/) page.
- Live detection results for ACM 2026 are at [/acm2026](/acm2026/).
- Questions or collaboration: [planetarypat@gmail.com](mailto:planetarypat@gmail.com).
